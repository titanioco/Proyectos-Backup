How to View Your Application

Since this is a GUI application running in a cloud environment, you need to access it via VNC:

  1. Port Forwarding (Recommended):
    • In VS Code, go to the "Ports" tab (usually next to Terminal)
    • Forward port 5900 for VNC access
    • Use any VNC viewer to connect to localhost:5900
  2. Web VNC (Alternative):
    • Forward port 6080 in VS Code
    • Open http://localhost:6080 in your browser
    • Use a web-based VNC client like noVNC
